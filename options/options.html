INSTRUCTIONS

Filtrer des annonces

  • Rendez vous sur un groupe facebook disposant de publications de ventes.
  • Rendez vous sur la page "Publications de vente". Vous pouvez également utiliser l'outil de recherche de facebook afin d'obtenir un jeu d'annonces pertinent
  • Assurez vous que l'extension est correctement configurée
  • Filtrez !

Astuce

L'extension filtre uniquement les annonces déjà affichées. Ainsi, lorsque vous souhaitez traiter un grand nombre d'annonces, il est conseillé d'en charger un maximum sur la page avant d'exécuter le script

Il est également possible de charger un certain nombre d'annonces, les filtrer, en charger à nouveau, etc ...

Configurer l'application

Afin de correctement paramétrer l'extension, dans la partie option, entrez les prix minimum et maximum dont vous souhaitez voir apparaître les annonces. Dès lors qu'un des prix est changé, la modification est prise en compte et les nouveaux paramètres sont appliqués instantanément
